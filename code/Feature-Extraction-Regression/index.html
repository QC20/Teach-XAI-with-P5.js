Non-Discrete Video Feature Extractor

This project demonstrates a Non-Discrete Video Feature Extractor using machine learning. It uses a MobileNet model for feature extraction and a regression model to predict continuous values based on video input. The visualization allows you to train the model on custom examples and see real-time predictions. Key features: - Real-time video processing - Feature extraction using MobileNet - Custom regression model training - Continuous value prediction Instructions: - Use the slider to set a value for the current video frame - Click "add example image" to add the current frame as a training example - After adding several examples, click "train" to train the model - Once trained, the model will continuously predict values based on the video input
