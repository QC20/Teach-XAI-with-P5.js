Perceptron Visualization

This visualization demonstrates a simple perceptron, a fundamental building block of neural networks. The perceptron learns to classify inputs based on training data. You can interact with the visualization to add training data, start/stop training, and make predictions. Key concepts illustrated: - Binary classification - Supervised learning - Weight adjustment - Iterative training process Instructions: - Press SPACE to add training data - Press 'T' to start or stop training - Press 'P' for a prediction - Click on circles to change input states
